## Установка

```bash
# Клонируем репозиторий
git clone https://github.com/abdurajabsabirov-arch/Behavior-Compass.git
cd Behavior-Compass

# Создаем виртуальное окружение
python -m venv .venv
.venv\Scripts\activate  # Windows
# или
source .venv/bin/activate  # Linux/Mac

# Устанавливаем зависимости
pip install -r requirements.txt

# Копируем .env файл (уже создан с токеном)
# На Windows: просто используйте существующий .env файл

# Запускаем бота (терминал 1)
python run_bot.py

# Запускаем админ-панель (терминал 2)
python run_admin.py
```

## Использование

1. **Запуск бота**: `python run_bot.py`
   - Бот начнет опрашивать (polling) сообщения
   - Откройте Telegram и найдите бота @Behavior_Compass_bot
   - Нажмите /start и следуйте инструкциям

2. **Запуск админ-панели**: `python run_admin.py`
   - Откройте http://127.0.0.1:8000 в браузере
   - Смотрите статистику, пользователей, результаты, отзывы, рефералов
   - Экспортируйте данные в Excel

## Функциональность

✅ **Telegram бот:**
- Выбор языка (5 языков)
- Регистрация (имя + телефон)
- Тест из 24 вопросов
- Расчет 4-цветного профиля
- Генерация PNG карточки результата
- Сбор обратной связи (рейтинг + комментарий)
- Реферальная система

✅ **Admin панель (FastAPI):**
- Dashboard с статистикой
- Просмотр пользователей
- Просмотр результатов тестов
- Просмотр отзывов
- Просмотр рефералов
- Экспорт данных в Excel (все таблицы)

## БД

Проект использует SQLite с SQLAlchemy ORM. База данных автоматически создается при первом запуске.

Таблицы:
- `users` - Пользователи
- `results` - Результаты тестов
- `feedback` - Отзывы
- `referrals` - Реферальная программа
